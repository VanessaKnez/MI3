Project: Find your way out

Find your way out is een spel waar je als speler je de vijanden niet maak aanraken en dat je naar de uitgang moet geraken.

Je hebt jouw vijaden (rode bolletjes) die een random positie krijgen om de x aantal seconden.Je mag jouw vijanden niet aanraken en je moet naar de uitgang gaan (groene vierkant). Eens je aan de uitgang bent moet je terug hetzelfde doen totdat je verliest. Elke keer dat je bij de uigang geraakt gaat jouw score omhoog.

Moeilijkste onderdelen

Collisie detecteren tussen de vijanden en de speler, de vijanden in hun scherm houden (ze kunnen ook buiten het scherm staan). Ik heb zo veel mogelijk hulp gevraagd aan mijn medestudenten maar die konden mij ook niet zo goed helpen. Ook heb ik me voor een aantal code onderdelen gebaseerd op het internet. Deze staan normaal gezien allemaal aangeduid in mijn code in commentaar.

Wat ook nog moeilijk was: functies voor de niveaus te kiezen, bewegen van speler met knoppen, bewegen van speler met buttons op het scherm, toestel standaard horizontaal zetten

Hoe ik het maakte

Om de collisie te detecteren met de vijanden en om de vijanden in hun schem te houden is mij niet gelukt. Ik heb wel een paar dingens geprobeerd.

Om de niveaus te heb ik variabele aangemaakt voor de zaken waarvan ik de waarden ging veranderen voor de niveau dat je koos (werken met variabelen was moeilijjk voor mij).

Voor het bewegen van de speler met de knoppen op de pc had ik een stuk code van het internet dat met een switch case werkte. Natuurlijk heb ik de code aangepast.

Bewegen van de speler met knoppen op het scherm wou wel lukken voor ik koos op welke niveau ik wou spelen, maar wou niet lukken nadat ik het niveau had gekozen. Dit heb ik niet echt kunnen oplossen.

Om de toestel standaard in een horizontale modus te zetten heb ik ook een stuk code vanop het internet genomen. Dit werkte ongeveer maar niet helemaal. Als jouw gsm in portrait mode staat toont die het spel horizontaal, maar eens dat je jouw toestel horizontaal zet is de CSS helemaal raar en klopt het niet echt meer.
